Shopee Margin Calculator

배송비 + 수수료 + 환율 + 희망마진 기준 판매가 역산 / 판매가 기준 검산
역산 모드: 희망마진 KRW 입력 → 국가별 권장 판매가 계산
검산 모드: 국가별 판매가 직접 입력 → 실제 예상이익 계산

상품 정보

적용무게 = max(실중량, 부피무게) 후 10g 올림
희망마진은 최소 보장 방식으로 계산

판매가 기준 빠른 비교

예: 경쟁자 판매가를 넣고 국가별 예상이익을 빠르게 비교할 때 사용
선택한 국가 판매가 기준으로 해당 국가 실제이익만 빠르게 확인
