Shopee Margin Calculator

배송비 + 수수료 + 환율 + 희망마진 기준 판매가 역산 / 판매가 검산

1) 계산 모드

역산 모드: 희망마진 KRW 입력 → 국가별 권장 판매가 계산
검산 모드: 국가별 판매가 직접 입력 → 실제 예상이익 계산

2) 상품 정보

적용무게 = max(실중량, 부피무게) 후 10g 올림

3) 국가별 설정

여기 값만 바꾸면 수수료/환율을 언제든 수정할 수 있어.
환율 기본값은 예시값이야. 네가 쓰는 실제 값으로 바꿔서 써.

5) 국가별 결과

국가 통화 적용무게(g) 해외배송비(현지) 해외배송비(KRW) 총수수료율 고정수수료 추가보정(KRW) 목표 판매가 정산예상(KRW) 예상이익(KRW)
정산예상(KRW)은 판매가에서 수수료 차감 후 KRW 환산한 값 기준.
예상이익(KRW) = 정산예상(KRW) - 기본원가(KRW) - 해외배송비(KRW) + 추가보정(KRW)
