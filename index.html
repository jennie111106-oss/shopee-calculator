Shopee Margin Calculator

배송비 + 수수료 + 환율 + 희망마진 기준 판매가 역산

1) 계산 모드

역산 모드: 희망마진 KRW 입력 → 국가별 권장 판매가 계산
검산 모드: 국가별 판매가 직접 입력 → 실제 예상이익 계산

2) 상품 정보

적용무게 = max(실중량, 부피무게) 후 10g 올림
KRW 및 TW/PH/VN/TH 판매가는 반올림 처리
⚙ 설정 접기/펼치기
기본 화면은 심플하게 두고, 환율/수수료는 여기서 관리하면 돼.

환율 설정

국가별 수수료 / 추가보정

환율/수수료 입력값은 자동 저장돼.

4) 국가별 결과

정산예상(KRW)은 판매가에서 수수료 차감 후 KRW 환산한 값 기준.
예상이익(KRW) = 정산예상(KRW) - 기본원가(KRW) - 해외배송비(KRW) + 추가보정(KRW)
